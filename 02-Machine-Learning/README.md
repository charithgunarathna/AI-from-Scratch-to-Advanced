Machine Learning (ML) is the science of teaching a computer how to make decisions or predictions based on data, without being explicitly programmed to do so. In this module, we will explore the core concepts and algorithms that power modern AI.

---

## 🔍 Main Learning Methods

1. **Supervised Learning:** Training a model using labeled data (where the correct answers are already provided). The model learns the relationship between the input data and the output.
   * *(Example: Predicting house prices or Medical diagnosis like cancer detection).*
2. **Unsupervised Learning:** Training a model using unlabeled data. The algorithm must figure out the hidden patterns, structures, and relationships on its own.
   * *(Example: Customer segmentation for marketing or Anomaly detection in banking).*
3. **Reinforcement Learning (RL):** Training a model to make a sequence of decisions by rewarding desired behaviors and punishing negative ones.
   * *(Example: AI learning to play chess, or Self-driving cars).*

---

## 🛠️ Core Algorithms

| Algorithm | Category | Common Use Cases |
| :--- | :--- | :--- |
| **Linear Regression** | Supervised (Regression) | Predicting continuous numbers like future stock prices, sales, or salary. |
| **Logistic Regression** | Supervised (Classification)| Binary classification (Yes/No tasks), such as Email Spam/Not Spam detection. |
| **Decision Trees** | Supervised (Both) | Making decisions based on feature splits. Great for risk assessment. |
| **Random Forest** | Supervised (Ensemble) | Combining multiple Decision Trees to get higher accuracy and prevent overfitting. |
| **Support Vector Machines (SVM)** | Supervised (Classification)| Finding the best optimal boundary (hyperplane) to separate different classes of data. |
| **K-Means Clustering** | Unsupervised (Clustering) | Grouping similar data points together (e.g., grouping website users by behavior). |

---

## 📈 Model Evaluation (Measuring Success)

Building a model isn't enough; you must prove it works.
* **For Classification Models:** We use metrics like Accuracy, Precision, Recall, and the F1-Score.
* **For Regression Models:** We use metrics like Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and the R² Score.
